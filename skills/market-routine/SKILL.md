# Market Routine

Use this skill to run a repeatable A-share scanning workflow.

## Scan order

1. Official disclosures
   - Exchange announcements
   - Company announcements
   - Regulatory notices
   - Company IR / investor relations pages

2. Fast news
   - 财联社
   - Major financial breaking-news feeds

3. Theme / industry-chain clue sources
   - 韭研公社
   - Theme communities
   - Timeline and expansion discussions

4. Broad verification
   - 东方财富
   - Supporting quote/news hubs

5. Sentiment / discussion
   - 雪球
   - 抖音 and other creator platforms

## Time-based workflow

### Pre-open
- Check overnight macro and major announcements
- Note confirmed catalysts first
- Scan fast-news summaries
- Scan theme/community sources for possible resonance themes
- Split notes into:
  - confirmed facts
  - candidate clues to verify
  - pure sentiment / noise

### Intraday
- Watch fast news and unusual moves
- Map price action to catalysts
- Use theme/community sources to see expansion paths
- Use broad portals to cross-check affected stocks, sectors, and related concepts
- Verify any high-heat narrative against a primary source when possible

### Post-close
- Review strongest themes, failed themes, and late reversals
- Check which narratives were confirmed by disclosures
- Record recurring high-signal and low-signal sources/accounts
- Save tomorrow's follow-up questions

## Rules

- Facts first, opinions second, sentiment last.
- Trace claims back to a primary source whenever possible.
- Distinguish theme discovery from trade confirmation.
- Keep a running list of high-signal and low-signal sources.
- Preserve failed or noisy leads too, so the same noise is easier to ignore next time.

## Output pattern

- Key facts
- Emerging themes
- Confirmed catalysts
- Rumor / noise watch
- Accounts worth tracking
- Next checks
