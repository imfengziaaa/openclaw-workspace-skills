# Doc Summarizer

Summarize long material without losing the point.

## Workflow

1. Identify the material and audience.
2. Skim for structure first.
3. Extract the main claims, decisions, action items, and caveats.
4. Rewrite in the shortest form that preserves usefulness.
5. Preserve exact wording only when it matters.

## Good summary shapes

- Executive summary
- Bullet key points
- Decisions made
- Action items
- Risks / caveats
