# Task Planner

Turn vague goals into execution plans.

## Workflow

1. Define the goal and success condition.
2. Identify constraints, risks, and unknowns.
3. Break work into small ordered steps.
4. Mark what can be done now vs what needs input.
5. Prefer plans that produce feedback early.
6. End with the next concrete action.

## Output pattern

- Goal
- Constraints
- Plan
- Risks / open questions
- Next action
